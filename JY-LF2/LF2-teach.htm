我電腦配備不夠力,還是跑不動小鈞試版

把系統色彩調成256色會順暢許多,但是圖片會有些瑕疵..

在桌面按右鍵　→　點選設定值　→  色彩品質調成256色   →   確定

XP調成方法: 在小鈞試版程式上按右鍵  →  相容性   → 將256色選項打勾 → 確定

讓 WindowsXP 跑LF2更順的方法

1.將色彩主題調成傳統配置

在桌面按右鍵  → 將主題調成傳統配置 → 確定

2.把視覺效果全部關掉      

在我的電腦右鍵 → 內容 → 進階 (效能設定) → 將視覺效果調成最佳較能 → 確定

醬子就能降低(視訊、實體))記憶體的用量，就可以達到LF2跑順的效果。
